# FAQ Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a `/faq` page covering the Augur fork for REP holders, surfaced as the first (urgent) hero menu item on the landing page.

**Architecture:** Static `src/pages/faq.astro` page. No content collection — content is hardcoded. Collapsible Q&A uses native `<details>`/`<summary>` (zero JS). Landing page hero and footer each link to the page.

**Tech Stack:** Astro 5, Tailwind v4 (global.css), native HTML `<details>`/`<summary>`, existing layout components (`Button`, `Pointer`, `SocialLinks`, `SectionHeading`, `Footer`).
